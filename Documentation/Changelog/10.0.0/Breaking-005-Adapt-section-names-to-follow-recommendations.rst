=======================================================
Breaking: Adapt section names to follow recommendations
=======================================================

Description
===========

Section names are recommended to be uppercase as well
as partials or layout names. This is a breaking change.

If you have customized templates please ensure you check
the casing of your section names after applying this patch.

Migrated section names:

- actions => Actions
- content => Content
- intro => Intro
- paginator => Paginator
